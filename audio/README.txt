NHẠC NỀN KỊCH TÍNH
===================

Để dùng bài nhạc kịch tính làm nhạc nền khi làm bài:

1. Đặt file nhạc (MP3) vào thư mục này với tên:  background.mp3

2. Nên chọn nhạc không bản quyền (royalty-free), ví dụ:
   - Pixabay Music: https://pixabay.com/music/
   - Tìm: "suspense", "dramatic", "thriller", "countdown", "tension"

3. Nếu không có file background.mp3, app sẽ tự dùng nhạc tổng hợp (drone + nhịp tim).

4. Có thể chỉnh âm lượng trong js/sounds.js:  _bgMusic.volume = 0.45  (0 = tắt, 1 = max)
